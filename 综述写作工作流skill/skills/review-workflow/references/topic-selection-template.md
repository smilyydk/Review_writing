# 选题模板

## 目标
在开始综述写作前，先把题目收缩成一个可执行、可检索、可证明证据充分性的研究范围。

## 建议输出结构

- 题目：
- 备选英文工作题目：
- 文章类型：
- 时间范围：
- 最终写作语言：
- 文献优先级：
- 优先数据库：
- 重点覆盖方向：
- 预期结果：
- 约束条件：

## 必须明确的约束

1. 不得虚构作者、题目、DOI、期刊、年份、结论和参考文献。
2. 明确区分正式发表文献与预印本。
3. 对关键事实、关键引用和核心结论进行人工核查。
4. 所有中间过程应写入文件，而不是只停留在聊天输出中。
5. 若目标是综述定稿，需提前定义最低有效参考文献数与证据达标规则。

## 选题收缩提示

优先把题目写成：
- 对象域 + 方法对象 + 任务范围 + 边界/问题意识

例如：
- “大语言模型在某领域文本挖掘中的应用综述”
- “某学科中基于 RAG 的知识组织与知识发现综述”

避免：
- 题目过宽，只写“某领域中的 AI 应用综述”
- 题目与证据池不匹配
- 标题强调“知识发现”，正文却主要是问答或摘要生成
