# 图表规划模板

## 图表的角色
图表不只是美化，而是承担：
- 定义范围
- 比较代表性工作
- 展示证据分布
- 可视化挑战与未来方向

## 推荐图表类型

### 图
1. 研究范围与任务分类框架图
2. 技术路径/方法演化图
3. 挑战与风险框架图
4. 未来方向路线图

### 表
1. 任务覆盖矩阵
2. 正文主证据文献表
3. 章节证据强弱总览表
4. 高风险参考文献核查清单
5. 支撑层任务 vs 知识发现层任务比较表

## 规划原则

- 每张图表都要有明确用途和落点章节
- 正文关键判断应能与至少一张表形成锚定关系
- 图表内容应随补文献同步更新，避免正文与图表脱节

## 易错点

- 只列图名，不写用途
- 图表太多但不服务论证
- 比较表没有对应正文解释
