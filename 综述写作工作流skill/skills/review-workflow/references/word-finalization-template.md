# Word 定稿模板

## 目标
把 Markdown 综述整理为课程论文或投稿前的 Word-ready 成稿。

## 必做动作

1. 删除过程性提示词
2. 删除不需要的英文摘要/英文关键词（若目标格式不要求）
3. 将正文引文统一为一种格式
4. 按正文首次出现顺序生成参考文献表
5. 检查标题层级、表格、段距与字体要求

## 推荐终稿流程

1. 以最终 Markdown 为唯一文本源
2. 清洗以下内容：
   - 图表插入提示
   - 参考文献插入说明
   - 过程性写作痕迹
3. 做引文编号化
4. 追加文末参考文献
5. 再导出 docx / rtf / 其他 Word 可读格式

## 验证清单

- 文中是否只剩编号式引文
- 文末参考文献顺序是否正确
- 是否还有“初稿/修改稿/插入位置”等痕迹
- Word 中表格是否正常展开
- 图表是否放在应在的位置

## 可脚本化部分

- label -> reference ID 映射
- reference ID -> numbered references
- 删除固定提示段
- markdown -> docx 结构化转换
