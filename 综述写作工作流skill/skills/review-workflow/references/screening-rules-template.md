# 纳入排除标准模板

## 主题边界
先写清楚：本综述的主体是什么，不讨论什么，哪些只作为背景保留。

## 纳入标准

文献通常应满足以下大部分条件：
1. 明确涉及目标模型或方法。
2. 明确应用于目标领域文本场景。
3. 与核心任务有直接关系。
4. 题录可核查。
5. 摘要或题录中能看到任务、方法、数据或评测信息。
6. 时间范围符合要求。
7. 若仅有摘要，必须明确标注。

## 排除标准

原则上排除：
1. 与主题方法无关。
2. 与目标领域或目标任务关联弱。
3. 只有宣传性描述，没有可复核研究设置。
4. 关键元数据无法核查。
5. 虽带有领域关键词，但实际与综述主体不匹配。

## 优先级分层建议

- A 类：核心证据，可作为正文主分析对象
- B 类：重要辅助证据
- C 类：背景与讨论支撑
- D 类：排除

## 风险提醒

- 预印本不能直接等同于正式版主证据
- DOI/年份/期刊冲突条目必须标记为待核查
- 不要把“可引用线索”误当作“可直接定稿引用”
