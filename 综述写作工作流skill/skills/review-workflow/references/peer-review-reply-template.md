# 审稿回复模板

## 目标
模拟投稿后的 reviewer comment -> response -> revision 闭环。

## 建议产物

1. 模拟审稿意见.md
2. 修改回复表.md
3. 投稿前问题清单.md

## 模拟审稿意见结构

- 总体评价
- 主要问题（Major comments）
- 次要问题（Minor comments）
- 建议结论（接收 / 小修 / 大修）

## 修改回复表建议列

- 审稿意见编号
- 审稿意见
- 修改/回复
- 对应文件
- 是否已完成

## 投稿前问题清单建议项

- 题目与正文匹配度
- 引文与参考文献一致性
- 图表是否真正服务论证
- 证据薄弱段落是否措辞保守
- 英文摘要/中文摘要/关键词是否符合目标格式
- 过程性痕迹是否已删净

## 回复原则

- 正面承认合理问题
- 明确说明已修改到哪里
- 对未完全解决的问题说明边界，而不是回避
