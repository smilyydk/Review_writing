# 证据抽取模板

## 目标
把筛选后的文献转成可追溯、可比较、可映射到正文的结构化证据层。

## 最低建议产物

1. 证据提取表
2. 事实核查记录
3. 阅读笔记目录
4. 证据层诊断与正文映射说明
5. 定稿引文达标检查表

## 证据提取表建议字段

- ID
- Authors_Year
- Title
- Task
- Method
- Data_Source
- Evidence_Level
- Main_Finding
- Limitations
- Fact_Checked
- Note_File
- Notes

## 事实核查记录建议字段

- ID
- Title
- Author_Check
- Year_Check
- Venue_Check
- DOI_or_PMID_Check
- Version_Status
- Risk_Level
- Final_Use_Status
- Remarks

## 阅读笔记最少内容

- 标题
- 作者
- 年份
- 来源
- DOI / PMID / URL
- 主任务
- 方法与模型
- 数据与评测
- 主要结果
- 优势
- 局限
- 可归入章节
- 事实核查边界

## 关键原则

- 摘要级整理必须明确标注
- 正式发表版本优先于预印本
- 高风险题录不得直接进入终稿主证据
- 证据层要能反向支撑正文关键判断

## 证据分层建议

- 正文主证据
- 辅助证据
- 高风险待核查证据

## 完成标准

不是“表里有很多行”就算完成，而是：
- 每条文献有明确用途
- 关键章节有足够主证据
- 最终可直接定稿引文数量达标
