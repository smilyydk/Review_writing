# 示例提示词

以下提示词可用于触发 `review-workflow` skill。

## 从零开始
- 帮我从零搭一个课程综述项目，记得全过程留痕。
- 我想写一篇某领域综述，请帮我先做选题、检索和证据表结构。
- 请把综述写作拆成阶段化工作流，并把每一步都落到文件里。

## 中途接管
- 我已经有候选文献表了，请帮我继续做筛选标准、标签体系和证据抽取。
- 我已经写了初稿，请帮我从证据可见性和批判性分析角度修订。
- 请把这篇综述从修改稿推进到定稿，并补齐引文与参考文献顺序。

## 终稿整理
- 帮我把 markdown 综述整理成 Word-ready 版本。
- 把正文里的文本型引文替换成编号引用，并在文末排参考文献。
- 帮我模拟审稿意见，并写修改回复表。

## skill 沉淀
- 把这次综述写作流程沉淀成可复用模板。
- 请基于这个项目整理一个综述写作 skill，不要覆盖原项目。
