# 发布与安装说明

## 1. 仓库用途
本仓库是一个独立的 Claude Code skill/plugin 仓库，用于复用“综述写作工作流”，不会覆盖原始项目。

## 2. 立即初始化 git 是什么意思
“初始化 git”就是把这个文件夹变成一个 Git 仓库，方便你：
- 记录版本历史
- 提交修改
- 推送到 GitHub

在该目录下运行：

```bash
git init
git add .
git commit -m "init review workflow skill"
```

如果你后续要上传到 GitHub，再继续：

```bash
git branch -M main
git remote add origin <你的GitHub仓库地址>
git push -u origin main
```

## 3. 推荐发布步骤

1. 在 GitHub 上新建空仓库
2. 本地进入本目录
3. 执行 git 初始化与首次提交
4. 绑定远程仓库并推送

## 4. Claude Code 中的组织方式

核心文件：
- `.claude-plugin/plugin.json`
- `skills/review-workflow/SKILL.md`

模板材料：
- `skills/review-workflow/references/`

案例与示例：
- `skills/review-workflow/examples/`

## 5. 后续维护建议

- 新增模板时优先放入 `references/`
- 新增真实项目案例时优先放入 `examples/`
- 如需脚本化自动化，再逐步补充 `scripts/`
- 尽量保持 `SKILL.md` 为高层入口，不把所有细节都堆进主文件
